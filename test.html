Typing Hell - Test Build

Press SPACE or ENTER to start the game, then start typing the displayed text.

Check the browser console (F12) for debug logs.

Controls:

  • SPACE/ENTER: Start game from menu
  • Type the displayed text to attack enemies
  • ESCAPE: Pause/Resume game
  • BACKSPACE: Delete last character

What to expect:

  • Text should appear with proper color coding
  • Typed characters should turn green
  • Current character should be highlighted
  • Completing words should launch attacks
  • Enemies should spawn and move toward center
  • Projectiles should hit enemies and destroy them
  • Game should not freeze when attacks hit
